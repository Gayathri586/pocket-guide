# POCKET GUIDE

A web-based tour guiding app prototype for India featuring:

- Splash screen with animated logo and "MADE IN INDIA"
- Auth (login/register) with OTP mock, national ID/passport fields, forgot password, permissions
- Persistent username in header, account page, terms & conditions
- Home with hero, 5-slide image carousel, India map watermark
- Transport planner with AI-suggested modes and prices
- Accommodation (luxury/budget) with AI suggestions
- Food suggestions by region and partner references (Swiggy, Zomato)
- Places to visit (temples, monuments, museums, beaches, malls, parks) with map markers
- Help centre chatbot for essentials and admin contact
- Exit flow prompting for review, reviews stored in localStorage
- Live location map via OpenStreetMap/Leaflet

## Run locally

```bash
npm install
npm run dev
```

Opens on http://localhost:5173
